# ROAD JUMPER

## ゲームの内容
障害物を飛び越え続けてハイスコアを目指す、横スクロールジャンプアクションゲーム

上手く飛び越えられず、左端に当たるとゲームオーバー

障害物が右から流れてくる（未達）

ゲームオーバーまでの時間でスコア増減（未達）
